## 四、支付接入（待营业执照）

### 流程

```
用户在 App 内点击"充值"
  → 选择金额（10/30/50/100元）
  → App 调用 server.py /api/payment/url 生成支付链接
  → 跳转微信/支付宝支付
  → 支付成功 → 商户后台回调 server.py /api/payment/callback
  → server.py 验证签名 → 自动加积分 → App 刷新余额
```
