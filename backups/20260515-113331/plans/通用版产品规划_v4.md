## 二、下一步开发计划

### Phase 6: 服务端部署 & 支付（1 周）P0

**目标**：让产品真正能收费

1. 部署 server.py 到云服务器（阿里云/腾讯云轻量应用服务器）
2. 配置 HTTPS 域名（api.heragent.io）+ Nginx 反向代理
3. DeepSeek API Key 放在服务端，客户端不持有 Key
4. 接入支付（面包多 / 微信支付），充值 → 自动加积分
5. 客户端与服务器联调

### Phase 7: Windows 版（1.5 周）P1

**目标**：覆盖 Windows 用户

1. 搭建 Windows 开发环境（Win 11 + Node.js + Python）
2. 适配 Hermes CLI 在 Windows 上的安装（PowerShell 安装脚本）
3. Electron 打包 Windows 版（NSIS 安装器）
4. Windows 特有适配：
   - 文件路径处理（反斜杠、盘符）
   - 窗口控制（最小化/最大化/关闭按钮位置）
   - 系统托盘
   - 通知系统
5. 测试完整流程：安装 → 注册 → 对话 → 文件处理 → 定时任务
6. 生成 Windows 安装包（.exe）

### Phase 8: 体验补完（1 周）P2

1. **思考过程展示** — 可折叠的步骤列表，显示耗时
2. **消息编辑** — 发送后可编辑自己的消息重新发送
3. **对话历史侧边栏** — 最近 N 条会话记录，可切换
4. **文件预览** — PDF/图片在对话中直接预览，不用下载
5. **应用更新** — 内置版本检查和在线更新（electron-updater）

### Phase 9: 规模上线准备（1 周）P2

1. **落地页优化** — 产品官网展示功能、定价、下载
2. **新手引导视频** — 2 分钟安装+使用教程
3. **错误监控** — Sentry 或其他错误追踪
4. **使用统计** — 匿名用量统计（DAU、功能使用率）
5. **CI/CD** — GitHub Actions 自动构建 macOS + Windows 安装包

## 三、Windows 版具体计划

### 技术适配

| 差异点 | macOS | Windows |
|--------|-------|---------|
| 安装路径 | ~/Library/Application Support/heragent/ | %APPDATA%/heragent/ |
| Hermes CLI | ~/.hermes/ | %USERPROFILE%/.hermes/ |
| Python | 系统自带 python3 | 需安装 Python 3.11+ |
| 终端 | bash/zsh | PowerShell / CMD |
| 窗口控制 | 红绿灯左上角 | 最小化/最大化/关闭右上角 |
| 系统托盘 | 原生支持 | 需适配 |
| 打包格式 | DMG / ZIP | NSIS .exe 安装器 |
| 构建 | electron-builder --mac | electron-builder --win |

### 工作量

| 任务 | 预估 |
|------|------|
| Windows 开发环境搭建 | 0.5 天 |
| Electron 主进程适配 | 1 天 |
| Hermes CLI 安装脚本（PowerShell） | 1.5 天 |
| 窗口/托盘/通知适配 | 1 天 |
| electron-builder 配置 + 打包 | 0.5 天 |
| 完整测试 + Bug 修复 | 2 天 |
| **总计** | **6.5 天** |
