# Hermes AI 通用版 · 产品规划

> 决策日期：2026-05-09
> 战略：通用版优先，快消版暂停。等通用版上线后回头做快消版。

---

## 一、定位

| 维度 | 快消版 | 通用版 |
|------|--------|--------|
| 用户 | 乳制品/冻品/酒水老板 | 所有人——学生、白领、自由职业、小老板 |
| 人格 | 干了七八年的老财务 | 聪明、靠谱、有礼貌的个人助理 |
| 场景 | 对账、催收、货损 | 写文章、读文件、搜信息、做表格、设提醒 |
| 卖点 | 帮老板多收10万回款 | **桌面上最聪明的助手，比网页版更方便，能读你的文件** |
| 定价 | 99元/年 | **99元/年**（不变，通用市场更大） |

一句话：**"比 ChatGPT 网页版多干一件事——能读你电脑上的文件，能帮你操作你的应用。"**

---

## 二、复用的部分（不用重新做）

| 组件 | 说明 |
|------|------|
| Hermes Agent 引擎 | 技能系统、工具集、定时任务、记忆——全部复用 |
| 积分计费服务 (server.py) | DeepSeek 代理转发、扣费、激活码——略改名就能用 |
| Electron 桌面壳 | 保持磨砂玻璃 UI、对话面板、报告中心——做减法不用做加法 |
| 安装器 + 打包脚本 | 保持 ZIP 分发 + install.sh |

**快消版的代码不是废的，是底座。通用版在上面做减法+改名。**

## 三、与快消版的具体差异

### 3.1 删掉的东西

- ❌ 6 个行业功能按钮（订单导入/对账/货损/催收/简报/工资试算）
- ❌ 快消品人格（老财务话术、舟谱术语、行业规矩）
- ❌ 所有快消专属技能（zhoupu-erp、financial-reports、dairy-pack profile）
- ❌ 催收看板、对账报告等行业报告模板

### 3.2 换成的东西

- ✅ 干净的对话主界面（像 ChatGPT，但更聪明）
- ✅ 3 个通用快捷入口（替代 6 个行业按钮）

| 快捷入口 | 干什么 | 举例 |
|----------|--------|------|
| 📄 读文件 | 上传文件让 AI 处理 | "帮我把这个 PDF 总结成 3 点" / "这个 Excel 表帮我筛选出大于100的行" |
| 🔍 搜信息 | 联网搜索 | "2026年新能源汽车销量排名" |
| ⏰ 设提醒 | 定时任务 | "每天早上9点把今天的天气推送到飞书" |

### 3.3 人格切换

快消版："张总，今天对账不平，差368.50元"

通用版："你好，文件分析完毕。这个合同有3个风险点需要注意……"

## 四、产品功能架构

```
Hermes AI 通用版
├── 💬 对话面板（主界面）
│   ├── 打字聊天
│   ├── 上传文件（PDF/Word/Excel/图片/代码）
│   ├── 快捷指令（/搜索 /读文件 /提醒）
│   └── 🎤 语音输入（v1.1）
├── 📄 报告中心（历史输出归档）
├── ⏰ 定时任务（cron 管理）
├── ⚙️ 设置
│   ├── API 密钥（DeepSeek / OpenAI / 其他）
│   ├── 消息推送（飞书/企微/微信）
│   ├── 积分与激活（余额/续费/到期日）
│   └── 在线升级
└── 🔌 技能市场（v2.0，让用户装插件）
```

## 六、定价与激活方案

**跟快消版一样，不改：**

- 30 天免费试用（每天 100 积分，够用）
- 99 元/年激活（5000 积分，正常够 3-4 个月）
- 续费 99 元/年不涨价
- API 费用用户自理（引导 DeepSeek 充值 10 元起）
- 面包多收款 → 自动发码 → 飞书私信

**积分差异说明：**
- 快消版：5000 积分花在大量对账/催收批量处理
- 通用版：5000 积分花在日常聊天/文件处理，用量更省
- 积分不够时 App 提示"余额不足，要不要续费？"

## 七、验证路径（最小跑通）

```
第一步：我自己用 3 天
   → 日常都通过通用版跟 Hermes 对话，不用飞书
   → 测试：读 PDF、搜新闻、写周报、设提醒

第二步：找 3 个非行业朋友试用
   → 最好有学生/白领/自媒体，不是快消圈的
   → 看他们会不会装、会不会用、愿不愿意付 99

第三步：根据反馈微调 → 正式上线
```

## 八、不做的事（避免跑偏）

- ❌ 不做 Windows/Linux 版（先 Mac，验证通过再说）
- ❌ 不做移动端 App（对话走飞书/企微就够了）
- ❌ 不做多语言（中文优先，足够大）
- ❌ 不做技能市场/插件商店（v2.0 的事）
- ❌ 不纠结 UI 设计（磨砂玻璃壳已经有，够好看了）
