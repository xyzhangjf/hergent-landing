## 一、定位

**HerAgent = 能操控电脑、自主进化的 AI 数字员工团队。**

| 维度 | 说明 |
|------|------|
| 用户 | 小白用户——不需要懂命令行、不会配 API、不想折腾 |
| 体验 | 下载 DMG → 拖进 Applications → 双击 → 开始用。3 步 |
| 人格 | 靠谱、利索、不啰嗦。能动手就不废话。 |
| 核心差异 | ① 能读你电脑文件、操作你的应用 ② 越用越懂你 ③ 角色分工团队 |
| 一句话 | **桌面上最聪明的 AI 团队，安装即用，越用越强。** |

- 功能持续更新，越用越聪明
- 10 元起步，小白不纠结

## 三、技术架构

### 系统架构

```
用户 Mac 电脑
├── HerAgent.app（Electron 桌面壳）
│   ├── 对话面板（主界面）
│   ├── 角色选择器（8 个 AI 角色）
│   ├── 定时任务管理
│   ├── 文件上传/拖拽
│   └── 积分余额 + 充值入口
│
├── Hermes CLI（本地 AI 引擎，首启自动安装）
│   ├── 工具执行（文件/浏览器/终端）
│   ├── 技能系统（按角色匹配）
│   ├── 记忆系统（越用越懂你）
│   └── 定时任务调度
│
└── HerAgent Cloud（远程服务端）
    ├── DeepSeek API 代理（持有 API Key）
    ├── 积分计费 + 扣减
    ├── 用户认证（手机号/微信）
    ├── 充值 + 账单
    └── 激活码系统
```

### 关键设计决策

| 决策 | 说明 |
|------|------|
| API Key 放服务端 | 安全 + 积分体系不可绕过。客户端是瘦终端，不持有任何 Key |
| Hermes CLI 本地跑 | 文件操作、浏览器操控、终端命令——这些必须本地执行 |
| 首启引导安装 | 首次启动自动安装 Hermes CLI，进度条展示，用户只需等待 |
| 渐进式权限 | 默认只聊天+读文件。删除/系统级操作按需弹框确认 |

## 九、不做的事

- ❌ 不做 Windows/Linux 版（先 Mac，验证通过再说）
- ❌ 不做移动端 App（对话可走飞书/企微，手机端不是核心场景）
- ❌ 不做多语言（中文优先）
- ❌ 不纠结 UI 重设计（磨砂玻璃壳已有，v1 只做减法）
- ❌ 不做纯飞书版（App 产品感 = 值钱感）
