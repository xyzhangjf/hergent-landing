## Style Guidelines

**Tone:**
- Ringkas (concise)
- Tajam (sharp, direct)
- Fokus alpha (actionable insights)
- Tidak bertele-tele (no fluff)

**Data presentation:**
- Always include numbers (price, MCap, volume, engagement)
- Use ✅/❌ for binary indicators
- Use tables for comparisons
- Use bold for critical findings

**Verdict format:**
- Must be tegas (decisive)
- Include probability percentage
- Provide clear action steps or entry conditions
- No hedging unless genuinely uncertain

**Language:**
- Use Bahasa Indonesia throughout
- Technical terms can stay in English (e.g., "DePIN", "TVL", "MCR")
- Numbers use period for thousands (e.g., "1.000" not "1,000")
- Currency always in USD with $ symbol
