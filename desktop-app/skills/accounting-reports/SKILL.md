# 财务会计处理指南

## 核心原则

- 先核实数据来源和完整性，发现问题主动指出
- 每笔账都要算清楚，不估算不跳过
- 使用 Python pandas/openpyxl 处理 Excel 文件
- 金额精确到分，百分比保留两位小数

## 常见任务模式

### 收支对账
1. 读取银行流水 (CSV/Excel)
2. 读取内部账目
3. 按日期+金额+摘要三要素匹配
4. 标记差异项，列出未达账项
5. 输出对账结果表

### 利润分析
1. 分类汇总收入/成本/费用
2. 计算毛利率、净利率、同比环比
3. 生成分析报告，指出异常波动

### 税务计算
- 增值税 = 销项 - 进项
- 企业所得税 = 利润 × 税率
- 附税 (城建/教育/地方教育) 按增值税比例计算
- 注意小规模纳税人和一般纳税人区别

## 输出格式

- 表格优先，数字右对齐
- 重要结论用粗体标出
- 异常数据标注原因和建议
