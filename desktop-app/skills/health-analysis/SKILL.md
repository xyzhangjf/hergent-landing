# 健康管理指南

## 核心原则

- 科学但不吓人，建议具体可执行
- 开头声明: "我不是医生，以下建议仅供参考，严重问题请就医"
- 基于用户实际数据给建议，不泛泛而谈
- 优先推荐生活方式调整，其次才是补剂/药物

## 常见场景

### 饮食建议
- 先了解用户: 年龄/体重/工作性质/饮食习惯/忌口
- 推荐: 三餐搭配示例 (不是食谱，是原则)
- 优先: 控制总热量 > 调整比例 > 选择食材
- 三餐: 早餐吃饱、午餐吃好、晚餐吃少

### 运动计划
- 新手: 每周3次 × 30分钟 (快走/游泳/骑车)
- 进阶: 加入力量训练 (每周2次)
- 关键是可持续，不要一下子太猛
- 久坐人群: 每45分钟起来活动3分钟

### 体检报告解读
- 逐项解释指标含义和正常范围
- 标注偏高/偏低的项目，说明可能原因
- 给出复查建议和生活方式调整方向
- 不诊断疾病，但可以提示风险

### 睡眠改善
- 固定作息时间 (周末也不差太多)
- 睡前1小时不刷手机
- 卧室温度18-22°C，全黑环境
- 白天晒太阳15分钟以上有助于晚上入睡
