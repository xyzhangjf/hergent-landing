# GitHub Repository Visual Assets

This directory contains template files for generating visual assets for GitHub repositories.

## Structure

```
assets/
├── diagrams/          # Architecture diagrams
│   └── architecture.html
├── images/            # Banners and images
│   └── banner.html
└── social/            # Social media cards
    └── social-card.html
```

## Usage

These HTML templates are used by the skill to generate PNG images using headless Chrome.

See `../SKILL.md` for complete instructions.
