# TikTok Analytics Terms: Indonesian ↔ English

## LIVE Analytics (Detail LIVE)

| Indonesian | English | Notes |
|-----------|---------|-------|
| Penonton | Viewers | Tab for viewer analytics |
| Keterlibatan | Engagement | Tab for engagement analytics |
| Rata-Rata Penonton | Average Viewers | Average concurrent viewers |
| PCU | Peak Concurrent Users | Same abbreviation in both languages |
| Menonton | Watching | Currently viewing (red line in graph) |
| Masuk | Entering | Viewers joining (blue line) |
| Keluar | Leaving | Viewers exiting (green line) |
| Sumber penonton | Viewer Sources | Traffic source breakdown |
| Feed Untuk Anda | For You Feed (FYP) | Algorithmic discovery |
| Tombol LIVE | LIVE Button | Direct LIVE notification/button |
| Tab Shop | Shop Tab | TikTok Shop discovery |
| Lainnya | Others | Other traffic sources |
| Profil pembeli | Buyer Profile | Demographics of buyers |
| Jenis kelamin | Gender | Male/Female split |
| Usia | Age | Age range demographics |
| Pengikut | Followers | Follower vs non-follower |
| Non-pengikut | Non-followers | Viewers who don't follow |
| Perempuan | Female | Gender category |
| Laki-laki | Male | Gender category |
| Sembunyikan grafik tren | Hide trend graph | Collapsible toggle |
| Lihat selengkapnya | See more | Expandable link |
