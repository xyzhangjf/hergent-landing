# 商务文档撰写指南

## 核心原则

- 结构清晰：标题→签约方→正文→签署栏
- 条款严谨：权利义务对等，无歧义表述
- 关键要素齐全：金额、期限、违约责任、争议解决
- 使用标准合同语言，避免口语化

## 常用合同模板结构

### 供货合同
1. 签约双方信息
2. 供货内容/规格/价格
3. 交货方式/时间/地点
4. 验收标准
5. 付款方式/账期
6. 违约责任
7. 争议解决方式

### 服务协议
1. 服务内容与范围
2. 服务期限
3. 费用与结算
4. 双方权利义务
5. 保密条款
6. 终止条件

## 写作要点

- 开篇明确签约方全称、地址
- 金额同时写大写和小写
- 日期精确到日
- 关键条款加标题便于检索
- 末尾预留签署栏 (甲方/乙方签字+日期)
