# 银行流水比对工具使用指南

## 📋 概述

这是一个自动化的银行流水比对工具，用于比对银行流水和财务系统流水，找出匹配项、差异项和缺失项。

## 🚀 快速开始

### 1. 准备数据文件

准备两个Excel或CSV文件：
- `银行流水.xlsx` - 银行导出的流水
- `财务系统流水.xlsx` - 财务系统导出的流水

### 2. 运行比对脚本

```bash
cd /workspace/projects/workspace

# 基本用法
python bank_reconciliation.py --bank 银行流水.xlsx --finance 财务系统流水.xlsx

# 指定输出目录
python bank_reconciliation.py \
  --bank 银行流水.xlsx \
  --finance 财务系统流水.xlsx \
  --output ./output
```

### 3. 查看结果

脚本会自动生成一个Excel报告，包含：
- 📊 统计汇总表
- 🔍 比对明细表
- 📁 原始银行流水
- 📁 原始财务系统流水

## 🔧 功能说明

### 1. 自动识别字段

脚本会自动识别以下关键字段：
- **金额** - 识别包含"金额"、"amount"、"money"等关键词的列
- **日期** - 识别包含"日期"、"date"、"时间"等关键词的列
- **摘要** - 识别包含"摘要"、"description"、"备注"等关键词的列

### 2. 数据清洗

- 统一列名格式（去除空格，转小写）
- 标准化金额格式（转换为数值）
- 标准化日期格式（转换为日期类型）
- 去除无效数据

### 3. 数据比对

- 按金额比对
- 统计匹配条数
- 标记银行多余或财务多余
- 计算缺失条数

## 🔍 常见问题

### Q: 脚本无法识别金额列怎么办？

A: 可以手动修改列名，包含"金额"、"amount"、"money"等关键词。

### Q: 比对结果不准确怎么办？

A: 
1. 检查两个文件的金额格式是否一致
2. 确保金额列没有特殊字符（如¥、$,等）
3. 检查日期格式是否统一

### Q: 可以处理多个月的数据吗？

A: 可以！脚本会处理所有数据，不分月份。建议按月分别比对。

### Q: 可以自定义比对规则吗？

A: 当前版本按金额比对。如需自定义规则，可以修改脚本中的 `reconcile()` 方法。
