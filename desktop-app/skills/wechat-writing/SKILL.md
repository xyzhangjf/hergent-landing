# 公众号 / 自媒体写作指南

## 核心原则

- 标题比内容重要 10 倍 (决定点开率)
- 开头第一句决定读者是否继续读
- 有观点、有态度，不写"正确的废话"
- 数据引用标明来源，增强可信度

## 文章结构

### 观点类文章
1. 标题: 制造冲突/悬念/利益点
2. 开头: 故事/数据/问题引入 (100字内)
3. 正文: 3-4个小标题展开
4. 结尾: 总结+互动引导

### 干货/教程类
1. 标题: "XXX终极指南/完整攻略"
2. 开头: 这篇文章能解决什么问题
3. 正文: 步骤1→2→3 (配截图/代码)
4. 结尾: 常见问题+资源链接

### 故事/传记类
1. 标题: 人物+核心冲突/转折
2. 开头: 高潮片段或时间锚点
3. 正文: 按时间线或主题展开
4. 结尾: 反思/启示/金句

## 写作要点

- 段落短 (手机屏3-5行)
- 每段一个核心信息
- 适当使用加粗、引用增强可读性
- 结尾加互动: "你怎么看？评论区聊聊"
